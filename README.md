# Objective  
Building an app demo by Streamlit on SinGAN

# Model Path  
```TrainedModels/lightning1/*.pth```

# Key Scripts  
1. ```backend.py```: compute animations
2. ```compute.py```: helper functions for computing animations
3. ```utils.py```: general helper functions
